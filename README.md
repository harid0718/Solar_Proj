# Solar Compliance Engine

## 🚀 Overview

A **fleet compliance monitoring system** that ingests solar PV performance data, evaluates sites against SLA contracts, and surfaces risks via an interactive dashboard.

Built to simulate real-world workflows of a **Data Analyst / Data Engineer in energy & operations**.

## 🧠 Key Capabilities

- End-to-end **data pipeline** from raw telemetry → analytics → dashboard  
- SLA evaluation across **uptime, energy, and incident response**  
- **Fleet risk scoring model** to prioritize high-risk systems  
- Interactive dashboard with:
  - Filtering (date, severity, system)
  - Drilldowns per site
  - Trend analysis
  - Incident tracking
  - Risk heatmap
